# CurrencyConverter
Application for converting different currency

## Branch
- Current branch - **main**

## Features
- Load list currencies
- Save loaded currencies
- Add currency to favorite
- Converting currencies

## Stack
- Swift
- UIKit
- VIPER
- Core Data
- URLSession
- UI programmatically (Constraints). Without Storyboard
- DI
- Unit and UI Tests
